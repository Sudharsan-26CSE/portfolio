Bank Management System

A brief description of what this project.

Key Features of Your System:

  • Account Management: Users can create new accounts, log in, and perform transactions securely.
  • Gift Card Integration: Supports gift card creation, top-up, and redemption.
  • Reward Points System: Allows users to earn and redeem points based on transactions.
  • Transaction History: Maintains a log of all user transactions.
  • Data Persistence: Saves account and transaction data to text files for continuity.
  • User Interface: Utilizes console-based UI with color-coded messages for better user experience.

Abstract

This project implements a comprehensive Bank Management System using Object-Oriented Programming (OOP) principles in C++. The system provides core banking functionalities such as account creation, login, deposit, withdrawal, and transaction history management. It extends traditional banking by integrating a Gift Card system, allowing users to generate, top up, and redeem gift cards, as well as a Reward Points mechanism where users earn and redeem points based on their transactions. The design leverages OOP concepts including encapsulation, inheritance, polymorphism, and abstraction. The Account base class manages user credentials, balances, and transaction logging, while derived classes like GiftCard and RedeemPoints add specialized features for gift card management and reward point redemption. Additional classes such as Notifier and various operation classes (e.g., DepositOperation, WithdrawOperation) demonstrate advanced OOP techniques, including virtual functions and abstract base classes. The user interface is console-based, featuring a splash screen, main menu, and user menu, with robust input validation and user notifications. All data, including account details and transaction histories, are persisted in text files to ensure continuity and traceability. The system is designed with security, scalability, and user experience in mind, and can be further enhanced with stronger encryption, improved error handling, and database integration for larger-scale deployments.
