# Personal Portfolio - Sudharsan

Welcome to my personal portfolio repository! This is a modern, responsive website built to showcase my skills, projects, and certificates as a passionate Web Developer.

## 🌟 Features

*   **Modern UI/UX:** Features a sleek dark theme with a glassmorphism bottom navigation bar.
*   **Smooth Animations:** Includes a dynamic typing effect, floating elements, and a multi-layered scrolling star background.
*   **Fully Responsive:** Carefully optimized for both desktop and mobile devices.
*   **Interactive Sections:** 
    *   About Me & Skills (with a downloadable resume)
    *   Horizontal scrolling Certificate gallery
    *   Project showcase with links to source code and live demos
*   **Functional Contact Form:** Integrated with Formspree for seamless and direct message delivery.

## 🛠️ Technologies Used

*   **HTML5**
*   **CSS3** (Custom properties, Flexbox, CSS Grid, Keyframe Animations)
*   **JavaScript** (Vanilla JS, Intersection Observer API for scroll animations)
*   **FontAwesome** (Icons)
*   **Google Fonts** (Poppins)

## 🚀 Getting Started

To view this project locally:

1. Clone this repository or download the files.
2. Navigate to the project folder.
3. Open `index.html` in your favorite web browser.
   *(Alternatively, you can use a local server like the VS Code "Live Server" extension for the best experience).*
